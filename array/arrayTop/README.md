# arrayTop

Use with typescript

```js
import { arrayTop } from 'https://raw.githubusercontent.com/ravid7000/elif/master/array/arrayTop/mod.ts'

const input = [1, 2, 3, 4, 5, 6]
arrayTop(input) // [1]
arrayTop(input, 3) // [1, 2, 3]
```

Use script

```html
<script src="https://raw.githubusercontent.com/ravid7000/elif/master/array/arrayTop/script.js"></script>
<script>
  var input = [1, 2, 3, 4, 5, 6]
  arrayTop(input) // [1]
  arrayTop(input, 3) // [1, 2, 3]
</script>
```
