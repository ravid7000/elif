# isArray

Use with typescript

```js
import { isArray } from 'https://raw.githubusercontent.com/ravid7000/elif/master/array/isArray/mod.ts'

const input = [1, 2]
if (isArray(input)) {
  // ...
}
```

Use script

```html
<script src="https://raw.githubusercontent.com/ravid7000/elif/master/array/isArray/script.js"></script>
<script>
  const input = [1, 2]
  if (isArray(input)) {
    // ...
  }
</script>
```
