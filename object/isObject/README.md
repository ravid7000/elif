# isObject Util

Use with Typescript

```js
import { isObject } from 'https://raw.githubusercontent.com/ravid7000/elif/master/object/isObject/mod.ts'

if (isObject({})) {
  // true
}

if (isObject([])) {
  // true
}

if (isObject(null)) {
  // false
}
```

Use with Javascript

```html
<script src="https://raw.githubusercontent.com/ravid7000/elif/master/object/isObject/script.js"></script>
<script>
  if (isObject({})) {
    // true
  }

  if (isObject([])) {
    // true
  }

  if (isObject(null)) {
    // false
  }
</script>
```
