# isPlainObject Util

Use with Typescript

```js
import { isPlainObject } from 'https://raw.githubusercontent.com/ravid7000/elif/master/object/isPlainObject/mod.ts'

if (isPlainObject({})) {
  // true
}

if (isPlainObject([])) {
  // false
}

if (isPlainObject(null)) {
  // false
}
```

Use with Javascript

```html
<script src="https://raw.githubusercontent.com/ravid7000/elif/master/object/isPlainObject/script.js"></script>
<script>
  if (isPlainObject({})) {
    // true
  }

  if (isPlainObject([])) {
    // false
  }

  if (isPlainObject(null)) {
    // false
  }
</script>
```
